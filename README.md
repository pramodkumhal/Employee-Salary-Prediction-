# 👨‍💼 Employee-Salary-Prediction

This project predicts the salary of an employee using a **Linear Regression** model.

### 📌 What it uses:
- Age  
- Gender  
- Degree  
- Experience (in years)  
- Job Title  

### 🔧 Tools:
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Scikit-learn  

### 🎯 Goal:
To predict employee salary based on their background and job role.
